Code Description
<!--...--> Html comments describe what a section of code does for other coders, or can be used to sign a portion of code.
<!DOCTYPE html> Doctype declares what type of document the file is to the browser.
<html> html is the language for describing the structure of web pages. It gives authors the means to publish online documents with headings, texts, tables, lists, photots, etc; retrieve online information via hypertext links, at the click of a button; design forms for conducting transactions with remote services, for use in searching for information, making reservations, ordering products,etc. include spread-sheets, video clips, sound clips, and other applications directly in their documents.
<head > head is an element that is a container for metadata
<<meta>> Defines metadata about an HTML document
<title> Specifies the title of the document. There can only be 1 in any document. The title is not part of the text of the document, but is a property of the whole document. It may not contain anchors, paragraph marks, or highlighting
<link> Defines the relationship between a document and an external resource
<script> Embed a client-side JavaScript script, either with a statement or by pointing to an external script file
<body> Literally, the body of the page where all the content -- headers, subheaders, etc. -- are housed.
<header> A header element contains navigation links or introductory content
<span> This is an element that is a generic inline container for phrasing content, which does not represent anything
<article> Defines an article
<nav> defines navigation links
<div>
<ul> Defines a bulleted list, used with <li>, which creates the items
<p> This element is used to define a paragraph

The exact rendering (indentation, leading etc.) is not defined and may be a function of other tags, style sheets, etc.
<h1> The first header, he was #1 like that guy from spongebob. the h1 tag indicates the first subheader. in terms of subheaders, think of it as the α subheader.
